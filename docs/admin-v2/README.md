# REPS Admin v2 Documentation Pack

This folder contains the draft product specification for REPS Admin v2.

The pack is intended to be committed into the repository under:

```text
/docs/admin-v2/
```

Admin v2 is designed as the operating system for running REPS: business dashboard, operations centre, member timeline, revenue, trust, support, content, system governance, design system and implementation roadmap.

Start with:

```text
00-docs-index.md
01-philosophy-and-information-architecture.md
```

Then move through the remaining documents in order.
